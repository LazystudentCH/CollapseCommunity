#### 技术方案：
- 数据库：mysql
- 缓    存：redis
- 前   端：el & vue
- 服务器： centos 7(腾讯云) & nginx
- 本地环境：go1.13
- 容器：Docker (多个mysql节点用docker模拟,因为只有一台服务器)
- 其他数据存储： 七牛云
